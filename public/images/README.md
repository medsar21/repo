# Images

Placez vos images dans ce dossier selon la structure suivante :

```
images/
├── hero-lions.jpg (1920x1080)
├── hero-equipes.jpg
├── stadium-bg.jpg
├── og-default.jpg
├── joueurs/
│   ├── hakimi.jpg
│   ├── ziyech.jpg
│   └── ...
├── equipes/
├── actualites/
├── produits/
└── drapeaux/
```

**Note**: Pour tester rapidement, vous pouvez utiliser des images de https://unsplash.com
